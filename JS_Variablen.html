primitive Datentypen

number, string, boolean, symbol, undefined, null

referenzierte Datentypen

object, array, function

Bei primitiven Datentypen wird der Wert der Variable in den Speicher geschrieben. Für referenzierte Variablen wird ein Zeiger bzw. eine Referenz auf das Objekt gespeichert.

Bei primitiven Datentypen wird der Wert direkt an eine Funktion übergeben. Änderungen an diesem Wert innerhalb der Funktion haben keine Auswirkungen auf die ursprüngliche Variable außerhalb der Funktion.
Bei referenzierten Datentypen wird die Referenz an die Funktion übergeben. Änderungen am Objekt innerhalb der Funktion wirken sich auf das ursprüngliche Objekt außerhalb der Funktion aus.

Typabfrage / typeof

typeof variableName

Gibt 'number', 'string', 'boolean', 'symbol', 'function', 'object' oder 'undefined' zurück.

Vor allem zur Typabfrage von primitiven Datentypen.

Typabfrage / instanceof

variableName instanceof Object/Array/Function

Gibt true oder false zurück.

Sowohl Arrays als auch Funktionen sind Instanzen von Object.

Vor allem zur Typabfragen von referenzierten Datentypen.

Typabfrage / Array.isArray()

Array.isArray([1, 2, 3]) // true

Typabfrage / isNaN()

isNaN(9) // false

Typkonvertierung / String

value.toString();
String(value);

Typkonvertierung / Number

Number(value);
parseInt(value, Zahlensystem/Basis);
parseFloat(value);

Typkonvertierung / Boolean

Boolean(value);

0, '', null, undefined, NaN werden zu false konvertiert.

Alles andere wird zu true konvertiert.

Typkonvertierung / Array

Array.from(value);
Array.of(value);
Array(value);

Array(value) versucht das Objekt in ein Array zu konvertieren.

Typkonvertierung / Object

Object(value);

Typkonvertierung / JSON <-> Strings

let newStr = JSON.stringify(newObj); // '{'name':'John','age':30}'
let newObj = JSON.parse(newStr); // { name: 'John', age: 30 }

Typkonvertierung / Arrays <-> Strings

let newStr = newArr.join(','); // '1,2,3'
let newArr = newStr.split(','); // [1, 2, 3]
