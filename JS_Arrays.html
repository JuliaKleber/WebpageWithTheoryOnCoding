Array

können Elemente unterschiedlichen Datentyps beinhalten

let arr = [wert1, wert2] Erstellen eines Arrays
arr1.concat(arr2) Verkettung von Arrays
arr[i] Zugriff auf i-te Element

arr.unshift(wert) Element am Anfang hinzufügen
arr.push(wert) Element am Ende hinzufügen
arr[arr.length] = wert Element am Ende hinzufügen
arr.splice(i, 0, a, b) Einfügen von a und b nach dem i-ten Element

arr.shift() Element am Anfang entfernen
arr.pop() Element am Ende entfernen
arr.splice(i, n) Löschen von n Elementen ab i-ten Index
arr.copyWithin(target, start, end) Kopieren und Einfügen von Elementen an andere Stelle des Arrays (target)

newArr = arr.splice(i, n) Array mit n Elementen ab i-ten Index erstellen, Elemente werden aus arr gelöschtt
newArr = arr.slice(-3) Kopieren der letzten drei Elementet
newArr = arr.slice(1, 5) Kopieren des 2. bis 5. Elements

arr.includes(el, startindex) true, wenn Element el in Array enthalten ist
arr.indexOf(el, startindex) Index des ersten Vorkommens des Elements oder -1 zurückgeben
arr.lastIndexOf(el, endindex) Index des letzten Vorkommens des Elements oder -1 zurückgeben
arr.find(el, startindex) Gibt das Element oder -1 zurück

arr.sort((a, b) => a - b) Sortieren in aufsteigender Reihenfolge
arr.sort((a, b) => b - a) Sortieren in absteigender Reihenfolge
arr.reverse() Array umkehren

arr.toString() Konvertierung zu einem String
arr.toLocaleString() Konvertierung zu einem String
arr.valueOf() Konvertierung zu einem String
arr.join('') Konvertierung zu einem String

array.forEach()

Es wird eine Funktion auf jedes Element des Arrays angewendet

array.forEach((element, index, arr) => {
    block;
});

array.map()

arrayNew = array.map((element) => {});

Es wird ein neues Array auf Grundlage eines alten Arrays erstellt
indem auf den Elementen des alten Arrays eine Funktion aufgerufen wird.

array.filter()

Es wird ein neues Array erstellt, das nur einen Teil der Elemente enthält aufgrund einer definierten Bedingung

const newArray = array.filter(function(currentValue, index, arr) {
    block;
});

Rückgabewert: true, wenn das Element beibehalten werden soll, ansonsten false

const numbers = [1, 2, 3, 4, 5, 6];
const evenNumbers = numbers.filter(num => num % 2 === 0);

array.reduce()

Es wird ein einzelner Wert aus einem Array erstellt

const newArray = array.reduce((accumulator, currentValue, index, arr) => {
    block;
    accumulator + currentValue;
});

reduceRight() wie reduce, die Elemente werden aber in umgekehrter Reihenfolge durchlaufen.

array.find()

Sucht nach erstem Element, dass eine Bedingung erfüllt.

const boolean = arr.find(element => element.length > 3)

array.every()

Überprüft, ob alle Elemente eine Bedingung erfüllen

const boolean = arr.every(element => element.length > 3)

array.some()

Überprüft, ob mindestens ein Element eine Bedingung erfüllt

const boolean = arr.some(element => element.length > 3)

Überprüfen, ob ein Array leer ist

arr.length === 0
