DOM-Baum

Der DOM-Baum besteht aus Knoten, Kindern und Geschwistern.
Das Ursprungselement ist das document-Element.

document.body.firstElementChild.nextElementSibling
document.body.lastElementChild.previousElementSibling

Damit kann jedes Element der Seite angesteuert werden.

class

HTML <h1 class="header1"><h1>
Javascript document.getElementByClassName("header1").innerText = "Überschrift1";
Javascript document.getElementsByClassName("header1").innerText = "Überschrift1";

getElementByClassName gibt das erste Element oder null zurück, falls es keins mit dem CSS-Selector gibt.
getElementsByClass gibt alle entsprechenden Elemente zurück.

id-Attribut

HTML <h1 id="header1"><h1>
Javascript document.getElementById("header1").innerText = "Überschrift1";

Gibt das Element mit der id "header1" zurück.

querySelector

Javascript document.querySelector(#idName);
Javascript document.querySelectorAll(.className);

querySelector gibt das erste Element oder null zurück, falls es keins mit dem CSS-Selector gibt.
querySelectorAll gibt alle entsprechenden Elemente zurück.

getElementsByTagName

Javascript document.getElementsByTagName("p");

Elemente können Anhand von Tagnames gefunden werden. Alle Elemente mit dem Tag werden zurückgegeben.

Text und Attribute ändern

document.getElementById("link").innerText = "codewars.com";
document.getElementById("link").innerHTML = "<strong>codewars.com</strong>";
document.getElementById("link").href = "codewars.com";
document.getElementById("link").textContent = "text";
document.getElementById("link").style.color = "blue";
document.getElementById("link").style.fontFamily = "arial";

Elemente hinzufügen und entfernen

const newEl = document.createElement("p");
const newText = document.createTextNode("text");
const elToInsert = newEl.appendChild(newText);
document.getElementById("whatever").appendChild(elToInsert);

const container = document.getElementById("container");
const element = document.getElementById("element");
container.removeChild(element);

Elemente hinzufügen und entfernen

document.querySelector("element").setAttribute("href", "http://google.com");
document.querySelector("element").removeAttribute("href");

Klassen hinzufügen und entfernen

button.classList.add("d-none");
button.classList.remove("d-none");
