useEffect

useEffect(() => {
    ...
    return () => {
        ...
    };
}, [variable]);

Wenn sich die Variable in den eckigen
Klammern ändert, wird useEffect ausgeführt.

Wenn die eckigen Klammern leer sind,
wird useEffect nur beim Montieren der Komonente ausgeführt.

Der Code vor return wird ausgeführt,
wenn die Komponente montiert wird
oder sich die Variable ändert.

Der Code hinter return wird ausgeführt,
bevor die Komponente aus dem DOM entfernt wird.
