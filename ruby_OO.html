Erstellung einer Klasse

class KlassenName
  attr_reader :instanz_variable_eins
  attr_writer :instanz_variable_zwei
  attr_accessor :instanz_variable_drei

  def initialize(parameter)
    @instanz_variable_eins = parameter_eins
    @instanz_variable_zwei = parameter_zwei
    @instanz_variable_drei = parameter_drei
  end

  def getter_methode
    @instanz_variable_eins
  end

  def setter_methode
    @instanz_variable_eins += @instanz_variable_eins
  end

  def setter_methode_zwei(parameter_name)
    @instanz_variable_eins = parameter_name
  end
end

Deklaration einer neuen Instanz

instanz_name = KlassenName.new(attribute)

Reader oder Getter-Methoden sind Methoden
mit denen Daten der Instanz gelesen werden können.

Writer oder Setter-Methoden sind Methoden
mit denen Daten der Instanz verändert werden können.

Wenn auf eine Methode aus dem Hauptprogramm
aus nicht zugegriffen werden soll (Kapselung),
stellt man ihr ein private voran.

Instanzen werden in lower_snake_case benannt.
Klassen werden in UpperCamelCase benannt.
Dateien, die Klassen enthalten, werden in UpperCamelCase benannt.

Mit attr_reader wird festgelegt, welche Variablen man aus dem Hauptprogramm aus lesen kann ohne dafür eine Methode schreiben zu müssen.

Mit attr_writer wird festgelegt, welche Variablen man aus dem Hauptprogramm aus verändern kann ohne dafür eine Methode schreiben zu müssen.

Mit attr_accessor wird festgelegt, welche Variablen man aus dem Hauptprogramm aus lesen und verändern kann ohne dafür eine Methode schreiben zu müssen.
