props

  • properties
  • unveränderlich
  • props helfen Daten zwischen den React-Komponenten auszutauschen
  • Zugriff: props.propName
  • props können überschrieben werden:
  • let component = <Component {...props} propName={"value"} />;
  • Es können Standardwerte für einzelne props mit der Methode defaultProps definiert werden:
  • component.defaultProps = {propName: value}

PropTypes

PropTypes oder Typescript sollten genutzt werden damit keine props vom falschen Typ an die Komponenten übergeben werden und somit Fehler verursacht werden.

import {PropTypes} from prop-types

Person.propTypes = {
    name: PropTypes.string.isRequired,
    age: PropTypes.number
};

Werte von Kindern an Eltern übergeben

const handleName = (value) => {
    setValue(value);
};

<Funktionsname onName={handleName} />
<Funktionsname onName={() => handleName(value)} />


const handleName = useCallback((value) => {
    setValue(value);
}, []);

<Funktionsname onName={handleName} />
<Funktionsname onName={() => handleName(value)} />


Die erste Option wird verwendet, wenn die Leistung nicht entscheidend ist.
Ansonsten wird die zweite Option verwendet.
