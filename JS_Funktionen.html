Funktion

const functionName = () => {
    block;
};

Übergabewerte werden an die Funktion übergeben. Parameter sind die Platzhalter in der Funktionsdefinition, die bei der Definition der Funktion verwendet werden, während Argumente die konkreten Werte sind, die bei einem Funktionsaufruf an die Funktion übergeben werden.

Mit return wird ein Wert an den Hauptteil des Programms zurückgegeben. Das ist der Rückgabewert.
