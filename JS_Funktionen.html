Funktion

functionName() {
    block;
    return;
}

Funktion

const functionName = () => {
    block;
};

Übergabewerte werden an die Funktion übergeben. Parameter sind die Platzhalter in der Funktionsdefinition, die bei der Definition der Funktion verwendet werden, während Argumente die konkreten Werte sind, die bei einem Funktionsaufruf an die Funktion übergeben werden.

Mit return wird ein Wert an den Hauptteil des Programms zurückgegeben. Das ist der Rückgabewert.

Bei primitiven Datentypen wird der Wert direkt an eine Funktion übergeben. Änderungen an diesem Wert innerhalb der Funktion haben keine Auswirkungen auf die ursprüngliche Variable außerhalb der Funktion.

Bei referenzierten Datentypen wird die Referenz an die Funktion übergeben. Änderungen am Objekt innerhalb der Funktion wirken sich auf das ursprüngliche Objekt außerhalb der Funktion aus.

Wenn eine Funktion als Argument an eine andere Funktion übergeben wird, wird sie ohne Klammern übergeben.

functionOne(functionTwo)

Mit Klammern würde das Ergebnis von functionTwo an functionOne übergeben.
