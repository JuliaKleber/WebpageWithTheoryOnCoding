for-Schleife

for (let i = 0; i < arr.length; i++) {
    block;
}

Wenn i vorher schon deklariert wurde und sich i im Block ändert,
muss nur die zweite Bedingung der for-Schleife angegeben werden.
for-Schleife für zusammengesetzte Datentypen

for (let wert of array/set/map) {
    block;
}

Die Variable wert ist jeweils eine Kopie des entsprechenden Elementes des Datentyps.
Das heißt, man kann über sie nicht den Inhalt des Datentyps verändern.
Den Schlüssel einer Map steuert man mit wert[0] und den Wert mit wert[1] an.

Mit break; wird der gesamte Schleifendurchlauf sofort abgebrochen.

Mit continue wird nur der aktuelle Durchgang der Schleife abgebrochen
und der nächste Durchgang beginnt.
for-Schleife für Arrays

arrayName.forEach((element) => {
    block;
});
