Reguläre Ausdrücke

\d = [0-9]
\D = [^0-9]
\w = [A-Za-z0-9_]
\W = [^A-Za-z0-9_]
\s = whitespace
\S = no whitespace

Reguläre Ausdrücke

{m, n}: m bis n Wiederholungen
*: 0 der mehr Wiederholungen
+: 1 oder mehr Wiederholungen
?: 0 oder 1 Vorkommen

Methoden für Reguläre Ausdrücke

regex.test("text") true, wenn regex in Text enthalten ist
text.match(regex) gibt Array mit Übereinstimmungen zurück oder null
text.replace(regex, "new") gibt String zurück in dem Übereinstimmungen durch anderen Text ersetzt wurden
text.split(regex) Text wird an Übereinstimmungen geteilt
regex.exec(text) gibt erste Übereinstimmung im text zurück (oder null) und andere Informationen
