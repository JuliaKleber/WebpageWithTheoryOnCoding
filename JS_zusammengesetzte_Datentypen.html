Array

können Elemente unterschiedlichen Datentyps beinhalten

let arr = [wert1, wert2]; Erstellen eines Arrays
arr[i] Zugriff auf i-te Element
arr[arr.length] = wert; Element am Ende hinzufügen
arr.push(wert); Element am Ende hinzufügen
arr.unshift(wert); Element am Anfang hinzufügen
arr.shift(); Element am Ende entfernen
arr.pop(); Element am Ende entfernen
arr.includes("text"); true, wenn "text" in Array enthalten ist
arr.splice(i, n); n Elemente von i an werden aus dem Array gelöscht.

array.map()

arrayNew = array.map((element) => {});

Es wird ein neues Array auf Grundlage eines alten Arrays erstellt
indem auf den Elementen des alten Arrays eine Funktion aufgerufen wird.

array.filter()

Es wird ein neues Array erstellt, das nur einen Teil der Elemente enthält aufgrund einer definierten Bedingung

const newArray = array.filter(function(currentValue, index, arr) {
    block;
});

Rückgabewert: true, wenn das Element beibehalten werden soll, ansonsten false

const numbers = [1, 2, 3, 4, 5, 6];
const evenNumbers = numbers.filter(num => num % 2 === 0);

noch einpflegen:

Suchen und Filtern von Elementen: Index eines Elements finden: fruits.indexOf("Banane") gibt den Index von "Banane" zurück. Ein neues Array erstellen, das Elemente basierend auf einer Bedingung filtert: fruits.filter(fruit => fruit.length > 5). Sortieren von Arrays:

Sortieren in aufsteigender Reihenfolge: fruits.sort(). Sortieren in absteigender Reihenfolge: fruits.reverse(). Verketten von Arrays:

Arrays verketten: let allFruits = fruits.concat(["Kiwi", "Erdbeere"]). Überprüfen von Arrays:

Überprüfen, ob ein Element im Array vorhanden ist: fruits.includes("Apfel") gibt true zurück, wenn "Apfel" im Array ist. Überprüfen, ob alle Elemente einer Bedingung entsprechen: fruits.every(fruit => fruit.length > 3). Transformation von Arrays:

Ein neues Array erstellen, indem Sie eine Funktion auf jedes Element anwenden: fruits.map(fruit => fruit.toUpperCase()). Reduzieren von Arrays:

Reduzieren (aggregieren) von Array-Elementen zu einem einzelnen Wert: fruits.reduce((accumulator, currentValue) => accumulator + currentValue). Entfernen von Elementen nach Index:

Verwenden Sie splice zum Entfernen von Elementen an einem bestimmten Index: fruits.splice(1, 2) entfernt 2 Elemente ab Index 1.
Map

Schlüssel-Wert-Paare

let map = new Map(); Map deklarieren
map.set(schlüssel, wert); fügt Paar hinzu
map.get(schlüssel); gibt Wert zu Schlüssel zurück
map.has(schlüssel); ist schlüssel enthalten?
map.delete(schlüssel); löscht Eintrag
map.clear(); löscht alle Einträge
map.size(); Anzahl der Paare
WeakMaps sind ähnlich wie Maps,
können aber nur Objekte aufnehmen.
