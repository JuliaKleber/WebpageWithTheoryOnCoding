Allgemeines Flexbox Grid

Grid Container

display: grid; Grid wird zur Anordnung der Elemente genutzt
grid-template-columns: Festlegung der Größe der Spalten
grid-template-rows: Festlegung der Größe der Reihen
  • grid-template-rows: 25% 25% 25% 25%;
  • grid-template-rows: repeat(4, 25%);
  • grid-template-columns: 96px 3em 40% 1fr;
  • grid-template-columns: 2fr 1fr 2fr;

grid-template: grid-template-rows / grid-template-columns;
grid-template: 50px 1fr / 1fr 2fr;

Grid Elemente: Spalten

grid-column-start: Linie, ab der das Element angezeigt werden soll
grid-column-end: Linie, bis der das Element angezeigt werden soll
grid-column: 2 / 4; Linien, zwischen denen das Element angezeigt werden soll
  • Wenn es n Spalten gibt, gibt es n+1 Linien, die sie begrenzen.
  • Die Linien werden von links nach rechts durchgezählt.
  • Wenn man von rechts nach links zählt, werden negative Zahlen startend bei -1 genutzt.
  • Man kann bei grid-column-start/end auch "span m" angeben um anzugeben über wie viele Spalten es sich
  • erstrecken soll.

Grid Elemente: Reihen

grid-row-start: Linie, ab der das Element angezeigt werden soll
grid-row-end: Linie, bis der das Element angezeigt werden soll
grid-row: 2 / 4; Linien, zwischen denen das Element angezeigt werden soll
  • Wenn es n Reihen gibt, gibt es n+1 Linien, die sie begrenzen.
  • Die Linien werden von oben nach unten durchgezählt.
  • Wenn man von unten nach oben zählt, werden negative Zahlen startend bei -1 genutzt.
  • Man kann bei grid-row-start/end auch "span m" angeben um anzugeben über wie viele Reihen es sich
  • erstrecken soll.

Grid Elemente: Area

grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end;

Grid Elemente: Reihenfolge

order: Standardwert ist 0.
Kann auch negative Werte annehmen.
