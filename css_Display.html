Allgemeines Flexbox Grid

Display

display: inline; mehrere Elemente pro Zeile
display: block; ein Element pro Zeile
display: inline-block; Inline-Element, dessen Größe angepasst werden kann.

display: none; wird nicht angezeigt
visibility: hidden; wird nicht angezeigt; eingenommener Platz wird freigehalten.

Block-Elemente

Vor und nach dem block-Element steht ein Zeilenumbruch.
Es kann eine Höhe und eine Breite für das Element festgelegt werden.
Erstreckt sich standardmäßig über die komplette Zeile.

Inline-Elemente

Es ist nicht möglich Höhe und Breite für das Element festzulegen.
Ist nur so breit wie auch der Text ist.
