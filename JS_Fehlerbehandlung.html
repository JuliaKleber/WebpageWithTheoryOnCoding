Syntaxfehler

Können bei Google Chrome unter "Weitere Tools" und dann "Entwicklertools" angezeigt werden. Dann kann man in der Menüleiste auf "Console" klicken.

"unexpected Identifier" lässt darauf schließen, dass der Fehler bereits früher passiert ist.
Laufzeitfehler

Treten unter Javascript selten auf.
Können mit try-catch aufgefangen werden.

try {
    Anweisungen;
}
catch(err) {
    alert(err.name);
    alert(err.message);
}

semantische bzw. logische Fehler

Auf Debugging kann unter Entwicklertools und "Sources" zugriffen werden. Dort kann man im Code Haltepunkte (breakpoints) einfügen indem man links auf die Zeilennummer klickt.
