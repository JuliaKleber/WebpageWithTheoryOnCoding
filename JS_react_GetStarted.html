Um ein React-Projekt zu starten werden über das Terminal im Verzeichnis des Projektes folgende Befehle eingegeben:

  • npm install -g create-react-app
  • (npx) create-react-app Projektname
  • cd Projektname
  • npm start
  • http://localhost:3000 öffnen

Der erste Schritt kann weggelassen werden, wenn npx genutzt wird.

Getting started

  • im Terminal unter src: rm -- App.test.js App.css logo.svg reportWebVitals.js setupTests.js
  • Im Verzeichnis public in der Datei index.html den <title> anpassen.
  • reportWebVitals(); und was dazu gehört kann aus index.js gelöscht werden.
  • Im Verzeichnis scr einen Ordner für die Komponenten erstellen.
  • Im Verzeichnis scr in der Datei App.js die Komponenten einbinden.
  • Im Verzeichnis scr in der Datei index.js wird die Komponente App.js eingebunden.
  • Im Verzeichnis scr in der Datei index.css werden die Styles eingebunden.
