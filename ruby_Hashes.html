Methoden für Hashes (CRUD)

hash[key] Zugriff auf Value, der zum Key gehört
hash[key] = value Value des keys überschreiben
hash[key] = value neues Key Value Pair erstellen
hash.delete(key) Key Value Pair entfernen

Methoden für Hashes

hash.key?(key) ist Key im Hash enthalten?
hash.keys gibt die Keys des Hashs als Array aus
hash.values gibt die Values des Hashs als Array aus
hash.each_key zum Iterieren über die Keys
hash.each_value zum Iterieren über die Values
hash.key(value) Schlüssel zum Wert


for-Schleife für Hashes

  Anweisungen
end

Falls man key oder value nicht benötigt:
