Methoden für Arrays (Zugriff auf Elemente)

arr[i] Zugriff auf i-te Element, Start bei 0
arr[0] Zugriff auf das erste Element
arr[-1] Zugriff auf das letzte Element
arr.first Zugriff auf das erste Element
arr.last Zugriff auf das letzte Element
Methoden für Arrays (Infos über Arrays)

arr.min gibt das kleinste Element zurück
arr.max gibt das größte Element zurück
arr.length Anzahl Elemente des Arrays
arr.size Anzahl Elemente des Arrays
arr.count Anzahl Elemente des Arrays
arr.count(element) Anzahl der Vorkommen von element im Array
index(element) gibt den ersten Index des Elements zurück
arr.include?(element) ist Element in Array enthalten?
Methoden für Arrays (Manipulation des Arrays)

arr[i] = element Element am Index i überschreiben
arr.push(element) Element hinzufügen
arr << element Element hinzufügen
arr[arr.length] = element Element hinzufügen
arr.insert(index, element) Element am Index einfügen
delete(element) Löscht alle Vorkommen des Elements
delete_at(index) Löscht Element mit dem index


Methoden für Arrays

arr.sort Elemente im Array sortieren, bei Strings erst Großbuchstaben
arr.sort.reverse Elemente im Array in umgekehrter Reigenfolge sortieren
arr.join(", ") gibt einen String zurück
+ zwei Arrays werden zu einem Array zusammengeführt
shuffle Reihenfolge der Elemente wird nach dem Zufallsprinzip verändert
sample(sample_size) Ein zufälliges Subarray mit der sample_size wird generiert
[[1, 3], [4, 2]][0][1] = 3


array.map

array_new = array.map do |element|
  ...
end

Es wird ein neues Array mit veränderten Elementen auf Grundlage eines alten Arrays erstellt indem auf den Elementen des alten Arrays eine Methode aufgerufen wird.
