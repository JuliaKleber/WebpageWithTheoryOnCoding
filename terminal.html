Bash-Befehle zum Anzeigen des Ordnerinhaltes

ls listet Dateien und Ordner im Verzeichnis, list
ls -a listet auch versteckte Dateien und Folder
ls -h inklusive Dateigröße
ls -l long list
ls -lh
ll

Bash-Befehle zu Ordnern

mkdir folder_name make directory
mkdir -p folder_name no error if existing
mv folder path/to/directory move
rmdir path/to/directory remove, entfernt Datei

rm path/to/file remove, entfernt Datei
rm -rf foldername entfernt komplettes Verzeichnis, f für "force"
cat file_name zeigt Inhalt der Datei an
