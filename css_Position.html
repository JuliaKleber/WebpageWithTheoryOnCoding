Positionierung

mögliche Werte: static, fixed, absolute, relative, sticky

position: static; default
position: absolute; mit top: left: bottom: right: wird der Abstand zum Rand des übergeordneten Elements angegeben. Fester Platz auf Seite.
position: relative; mit top: left: bottom: right: wird das Element von der Stelle wo es mit static Anordnung stehen würde verschoben.
position: fixed; bleibt an derselben Stelle, wenn Nutzer scrollt

float and clear

float: left/right;

Kann einem Bild zugewiesen werden,
damit es im Text entsprechend eingebettet wird.

clear: left, right, both;

Damit kein anderes Element daneben positioniert wird.
