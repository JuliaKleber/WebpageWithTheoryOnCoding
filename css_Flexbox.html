Allgemeines Flexbox Grid

Container

display: flex; Flex wird zur Anordnung der Elemente genutzt. Der komplette Container wird genutzt.
flex-direction: column/row; Anordnung in einer Spalte oder in einer Reihe
flex-wrap: wrap; Elemente werden in verschiedenen Reihen oder Spalten angezeigt.
justify-content: center; horizontale Ausrichtung
align-items: center; vertikale Ausrichtung
align-content: center; Anordnung von Zeilen
flex-flow: column wrap; Kombiniert flex-direction und flex-wrap

flex-direction

flex-direction: row; Ausrichtung in einer Reihe
flex-direction: row-reverse; umgekehrte Ausrichtung in einer Reihe
flex-direction: column; Ausrichtung in einer Spalte
flex-direction: column-reverse; umgekehrte Ausrichtung in einer Spalte

flex-wrap

flex-wrap: wrap; Elemente werden in verschiedenen Zeilen angezeigt.
flex-wrap: nowrap; Alle Elemente sind in einer Zeile.
flex-wrap: wrap-reverse; Reihenfolge wird umgedreht. Die Elemente werden in verschiedenen Zeilen angezeigt.

justify-content

justify-content: flex-start; Ausrichtung links
justify-content: flex-end; Ausrichtung rechts
justify-content: center; Ausrichtung mittig
justify-content: space between; Ausrichtung mit identischem Abstand zwischen den Elementen
justify-content: space-around; Ausrichtung mit identischem Element-Margins

align-items

align-items: flex-start; Ausrichtung oben
align-items: flex-end; Ausrichtung unten
align-items: center; Ausrichtung mittig
align-items: baseline;
align-items: stretch;

align-content

align-content: flex-start; Ausrichtung oben
align-content: flex-end; Ausrichtung unten
align-content: center; Ausrichtung mittig
align-content: space-between; Ausrichtung mit identischem Abstand zwischen den Zeilen
align-content: space-around; Ausrichtung mit identischem Abstand zwischen den Elementen
align-content: stretch;

Elemente

align-self: Kann dieselben Werte wie align-items annehmen.
order: 1; Standardwert ist 0. Kann auch negative Werte annehmen.
flex: 1; Durch unterschiedliche Zahlen können die Größenverhältnisse der Elemente zueinander dargestellt werden.
